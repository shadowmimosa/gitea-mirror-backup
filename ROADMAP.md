# Gitea Mirror Backup 路线图

本文档记录项目的未来发展计划和功能规划。

---

## 🎯 当前版本

**v1.4.1** (2026-01-28)
- ✅ Web 管理界面
- ✅ 实时仪表板
- ✅ 快照批量管理
- ✅ Markdown 报告查看
- ✅ Docker 容器化部署

---

## 🚀 短期计划

### v1.5.0（计划中）

**核心功能**
- [ ] 实时备份任务监控
  - 显示当前正在运行的备份任务
  - 实时进度显示
  - 任务日志查看
  
- [ ] 备份任务手动触发
  - Web 界面一键触发备份
  - 支持单个仓库备份
  - 支持全量备份
  
- [ ] 用户管理功能
  - 添加/删除用户
  - 角色权限管理（管理员/只读用户）
  - 用户操作日志

**界面优化**
- [ ] 更多统计图表
  - 备份趋势图（时间序列）
  - 磁盘使用趋势
  - 仓库增长趋势
  
- [ ] 备份配置在线编辑
  - Web 界面编辑 config.yaml
  - 配置验证和预览
  - 配置历史版本管理

**通知系统**
- [ ] 邮件/Webhook 通知配置界面
  - 可视化配置通知渠道
  - 测试通知功能
  - 通知模板自定义

### v1.6.0（计划中）

**监控集成**
- [ ] Prometheus 监控集成
  - 导出备份指标（成功率、耗时、大小等）
  - 自定义告警规则
  
- [ ] Grafana 仪表板模板
  - 预制的可视化面板
  - 备份健康度监控
  - 异常检测可视化

**任务调度**
- [ ] 备份任务调度管理
  - Web 界面配置 cron 表达式
  - 多个定时任务支持
  - 任务优先级设置

**国际化**
- [ ] 多语言支持
  - 英文界面

  - 可扩展的语言包
